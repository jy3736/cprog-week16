## 輸出一維陣列的『指定』內容。

- 設計一個可輸出指定陣列區間 [s, e) 的函數。
- 指定區間選擇範例： 
    - 展示用陣列：int d[7];
    - d[0, 5) => d[0], d[1], d[2], d[3], d[4]
    - d[3, 6) => d[3], d[4], d[5]
    - d[0, -1) => d[0], d[1], d[2], d[3], d[4], d[5], d[6]    
- 資料輸出至少要有一個空格作為間隔。
- 建議的函數原型：
```C++
void dump(int arr[], int len, ...);
    - arr 參數指向一維陣列
    - len 給定的陣列長度
    - s 開始索引，內定值為 0
    - e 結束索引，內定值為 -1(表示全部)
```
